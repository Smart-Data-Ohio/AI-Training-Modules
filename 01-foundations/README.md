# Foundations

> The on-ramp. Everything you need to know before you start building with AI.

## Who This Is For

Developers who know how to code but haven't worked with AI/ML beyond casual use of ChatGPT. If you can write a for loop and read an API doc, you're ready.

No math background required. No ML experience assumed. We'll build the intuition first and get technical when it matters.

## The Modules

Work through these in order — each one builds on the last.

| # | Module | What You'll Learn | Time |
|---|--------|-------------------|------|
| 01 | [LLM Fundamentals](01-llm-fundamentals/) | How language models actually work — tokens, context windows, temperature, and why models hallucinate | ~1.5 hours |
| 02 | [Prompt Engineering](02-prompt-engineering/) | How to write prompts that produce reliable, useful outputs — structured prompts, few-shot, chain-of-thought | ~1.5 hours |
| 03 | [AI-Assisted Coding](03-ai-assisted-coding/) | Copilot, Claude Code, Cursor — what they're good at, what they're bad at, and when to trust them | ~1.5 hours |
| 04 | [Ethics & Liability](04-ethics-and-liability/) | Client data, IP ownership, bias, and the stuff that keeps you out of trouble | ~1 hour |

**Total estimated time: 4–6 hours**

## Why Start Here

You can't make good decisions about AI tools if you don't understand how they work under the hood. These four modules give you the mental models you need to:

- Scope AI features accurately (instead of guessing)
- Set realistic expectations with clients
- Choose the right tool for the job
- Avoid the mistakes that make AI projects fail

## What's Next

After completing Foundations, move on to [02-Applied](../02-applied/) to start building real things with LLM APIs, or jump to [03-Classical ML](../03-classical-ml/) if you want to learn when an LLM isn't the answer.
